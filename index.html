You can use wethod integrations to automate your workflows and connect wethod with the tools you already use. There are two ways to integrate with wethod:

REST API​

Build custom scripts and applications that interact directly with your wethod data. The REST API gives you full programmatic access to resources like projects, budgets, invoices, timesheets, and more.

This is ideal if you are a developer looking to create tailored integrations or automate complex processes.

Get started with the REST API

Zapier​

Connect wethod to 7,000+ apps without writing any code. With Zapier, you can set up automated workflows that trigger actions across your favorite tools whenever something happens in wethod.

This is ideal if you want quick, no-code automation - for example, sending a Slack notification when a new project is created, or adding a row to a Google Sheet when an invoice is sent.
